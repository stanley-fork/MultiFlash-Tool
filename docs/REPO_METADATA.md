# 仓库元数据配置建议 (Repository Metadata)

请在 GitHub 和 Gitee 仓库的 **"About" (关于)** 设置中使用以下信息。完善这些信息有助于用户搜索到您的项目。

## 📝 Description (项目描述)

建议使用简洁明了的双语描述：

**中文 (Gitee 推荐)**:
```text
一款开源的多功能安卓刷机工具，支持高通 EDL (9008) 和 Fastboot 模式。高效、稳定、易用。支持 Payload 提取、分区管理、救砖和多设备操作。
```

**英文 (GitHub 推荐)**:
```text
An open-source multi-function Android flashing tool supporting Qualcomm EDL (9008) and Fastboot modes. Features Payload extraction, partition management, unbrick, and multi-device support. Efficient, stable, and easy to use.
```

## ⚙️ 如何设置

### 在 GitHub 上设置
1. 进入仓库主页：[https://github.com/xiriovo/MultiFlash-Tool](https://github.com/xiriovo/MultiFlash-Tool)
2. 点击右侧边栏 **About** 旁边的 **⚙️ (齿轮图标)**。
3. 在 **Description** 中粘贴上面的英文描述。
4. 在 **Website** 中粘贴链接。
5. 在 **Topics** 中输入标签（如 `android`, `edl` 等），按回车确认。
6. 点击 **Save changes**。

### 在 Gitee 上设置
1. 进入仓库主页：[https://gitee.com/xiriery778/MultiFlash-Tool](https://gitee.com/xiriery778/MultiFlash-Tool)
2. 点击仓库名称下方的 **编辑** 按钮（或进入"管理" -> "基本信息"）。
3. 填写 **项目简介**（使用上面的中文描述）。
4. 填写 **项目标签**。
5. 点击 **保存**。
